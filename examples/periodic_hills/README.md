# Periodic hills geometry examples

All cases in `cases/` are runnable with the single `periodic_hills` binary.

### Run

From the repo root:

```bash
./examples/periodic_hills/run.sh hills_laminar
./examples/periodic_hills/run.sh hills_baseline
./examples/periodic_hills/run.sh hills_sst
```

`run.sh` automatically writes results to `examples/periodic_hills/output/<case>/`.
