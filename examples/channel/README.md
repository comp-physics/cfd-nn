# Channel geometry examples

All cases in `cases/` are runnable with the single `channel` binary.

### Run

From the repo root:

```bash
./examples/channel/run.sh poiseuille_steady
./examples/channel/run.sh retau180_baseline
./examples/channel/run.sh retau180_sst
./examples/channel/run.sh retau180_unsteady_laminar
```

`run.sh` automatically writes results to `examples/channel/output/<case>/`.
